## 功能一览

### 屏幕截图

- 支持全屏、自定义区域、窗口截图
- 多显示器支持，自动适配 HiDPI / 高分屏
- 智能窗口识别 — 鼠标悬停自动吸附窗口/控件边界，无需手动对齐
- 截图历史记忆 — 随时一键重新截取上次区域

### 屏幕录制

- 自定义区域录屏，输出 H.264 编码视频
- 可配置帧率 (1–60 FPS) 与码率
- 支持同时录制麦克风和系统音频
- 录制过程中可暂停/恢复
- 3 秒倒计时开始，从容准备

### 标注工具

- **图形标注**：矩形、椭圆、线条、箭头
- **手绘标注**：铅笔自由绘制
- **文字标注**：添加文字说明
- **隐私保护**：马赛克、高斯模糊
- **撤销/重做**：随时修改，不怕画错

### 贴图（浮动窗口）

- 将截图、文字、剪贴板内容固定在屏幕最上层
- 支持调整透明度、大小、位置
- 可同时贴多张图，方便对照参考
- 截图时可排除贴图窗口，避免干扰

### 颜色拾取

- 屏幕任意位置取色
- 像素放大镜，精确定位像素
- 支持 RGB / HSV / HEX 多种颜色格式互转

### 水印

- 文字水印 — 自定义文本、位置与透明度
- 图片水印 — 叠加 Logo 或签名图片

### 其他特性

- 系统托盘常驻，随时快捷调用
- 支持开机自启动
- 内置更新检查，自动提示新版本
- 剪贴板智能读取（图像/文本）

## 快捷键

| 快捷键 | 功能 |
|:------:|------|
| **F1** | 截图（自动识别窗口 / 手动拖选区域） |
| **F3** | 将截图贴到屏幕 |
| **Esc** | 取消当前操作 |

> 更多快捷键可在设置中自定义。

## 安装

1. 前往 [Releases](https://github.com/Yangqihu0328/loong-pixelgrab/releases) 页面下载最新安装包
2. 运行安装程序，按提示完成安装
3. 启动 PixelGrab，托盘图标出现即可开始使用

### 系统要求

- **Windows** 10 / 11（64 位）
- macOS、Linux 版本开发中
