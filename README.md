## 功能特性

| 功能 | 说明 |
|------|------|
| 智能截图 | 自动识别窗口边界，也可手动拖拽选区 |
| 实时标注 | 矩形、椭圆、箭头、线条、文字、马赛克、模糊 |
| 屏幕贴图 | 将截图/标注结果钉在屏幕最前方，随时参考 |
| 剪贴板复制 | 一键复制标注结果到剪贴板 |
| 系统托盘 | 后台运行，不占桌面空间 |
| 自定义热键 | 可在设置中修改截图和贴图的快捷键 |
| 开机自启 | 可选开机自动启动 |
| 自动更新 | 启动时检测新版本，一键跳转下载 |

## 安装

### 方式一：安装包（推荐）

1. 在 [Releases 页面](https://github.com/Yangqihu0328/loong-pixelgrab/releases/latest) 下载 `PixelGrab-x.x.x-win64.exe`
2. 双击运行安装程序
3. 安装完成后勾选"运行 PixelGrab"即可启动

### 方式二：便携版

1. 下载 `PixelGrab-x.x.x-win64.zip`
2. 解压到任意目录
3. 双击 `bin\PixelGrab.exe` 运行

## 使用说明

### 快速开始

安装后 PixelGrab 会在系统托盘（屏幕右下角）显示图标，通过快捷键或右键托盘菜单操作。

### 快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| **F1** | 截图 | 进入选区模式 |
| **F3** | 贴图 | 将上次截图钉到屏幕 |
| **Esc** | 取消 | 取消当前截图或标注 |

> 快捷键可在 托盘右键 → 设置 中自定义修改

### 截图流程

```
按 F1 → 鼠标移动自动识别窗口 → 点击确认 / 拖拽自定义选区
                                    ↓
                              进入标注模式
                                    ↓
                    使用工具栏标注 → 贴图 / 复制 / 取消
```

**详细步骤**：

1. **按 F1** 进入截图模式，屏幕会出现蓝色边框高亮当前窗口
2. **点击**确认自动识别的窗口，或**拖拽**选择自定义矩形区域
3. 进入标注模式后，使用顶部工具栏添加标注：

### 标注工具

| 工具 | 说明 |
|------|------|
| 矩形 | 绘制矩形框 |
| 椭圆 | 绘制椭圆 |
| 箭头 | 绘制箭头，用于指示 |
| 线条 | 绘制直线 |
| 文字 | 在指定位置输入文字 |
| 马赛克 | 对选区打马赛克，用于遮挡敏感信息 |
| 模糊 | 对选区进行高斯模糊 |

### 标注操作

| 操作 | 方式 |
|------|------|
| 绘制形状 | 在画布上拖拽鼠标 |
| 输入文字 | 选择文字工具 → 点击画布 → 在弹出框输入 |
| 撤销 | 点击工具栏撤销按钮，或右键 |
| 重做 | 点击工具栏重做按钮 |

### 完成标注后

| 按钮 | 功能 |
|------|------|
| **贴图** | 将标注结果钉在屏幕最前方（可作为参考浮窗） |
| **复制** | 复制标注结果到剪贴板，可直接粘贴到微信、Word 等 |
| **取消** | 放弃本次截图和标注 |

### 贴图操作

- **F3**：将上次截图/标注结果贴到屏幕
- **双击贴图**：关闭贴图窗口
- 贴图会保持在所有窗口最前方，方便参考

### 系统托盘菜单

右键点击托盘图标：

| 菜单项 | 功能 |
|--------|------|
| 截图 | 开始截图（同 F1） |
| 贴图 | 贴图（同 F3） |
| 设置 | 修改截图/贴图快捷键 |
| 开机自启 | 开启/关闭开机自动启动 |
| 检查更新 | 手动检查是否有新版本 |
| 关于 | 查看版本信息、检查更新 |
| 退出 | 退出 PixelGrab |

## 更新

PixelGrab 启动时会自动检查新版本。你也可以通过以下方式手动检查：

- 托盘右键 → **检查更新**
- 托盘右键 → **关于** → 点击"是"

发现新版本后，点击"是"会自动打开浏览器跳转到下载页面。
